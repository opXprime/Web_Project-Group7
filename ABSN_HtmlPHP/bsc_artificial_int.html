BSc Artificial Intelligence Modules

Year 1 (Core Modules)
Module ID Module Name Module Leader ID Description
1 Introduction to Programming 1 Covers the fundamentals of programming using Python and Java.
2 Mathematics for Artificial Intelligence 2 Teaches discrete mathematics, calculus, linear algebra, and probability theory for AI.
3 Computer Systems & Architecture 3 Explores CPU design, memory management, and assembly language.
4 Data Structures and Algorithms 4 Introduction to algorithms and data structures, focusing on their implementation in AI applications.
Year 2 (Increasing Specialization)
Module ID Module Name Module Leader ID Description
5 Introduction to Artificial Intelligence 5 Fundamentals of AI, including search algorithms, problem-solving, and agent-based systems.
6 Machine Learning 6 Covers supervised and unsupervised learning, neural networks, and clustering algorithms.
7 Computer Vision 7 Studies image processing, object detection, and deep learning for visual data analysis.
8 Natural Language Processing 8 Introduction to language models, text analysis, and NLP applications for AI systems.
Year 3 (Advanced Topics & Final Year Project)
Module ID Module Name Module Leader ID Description
9 Deep Learning 5 Advanced neural network architectures, backpropagation, and applications in AI.
10 Reinforcement Learning 6 Focus on learning through rewards and penalties, Markov Decision Processes, and Q-learning.
11 Robotics 7 AI applications in robotics, including control systems, navigation, and robot perception.
12 Ethics in Artificial Intelligence 8 Explores the ethical considerations of AI, including bias, fairness, and accountability.
13 Final Year Project 9 A major independent project where students develop an AI solution to a real-world problem.
